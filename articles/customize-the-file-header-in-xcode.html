Customize the file header in Xcode

Xcode
January 27, 2025
Tired of repetitive file header edits in Xcode? Meet IDETemplateMacros.plist. This file allows developers to define default settings and values for newly created files in Xcode projects, enabling consistency and saving time.
What is IDETemplateMacros.plist?
IDETemplateMacros.plist is a property list file that Xcode uses to populate predefined values in new file templates. For instance, when you create a new Swift file, Xcode automatically inserts placeholders for the file's author, company name, and copyright notice.

By customizing IDETemplateMacros.plist, you can control these default values and make them reflect your project's needs.
Why use IDETemplateMacros.plist?
Consistency: Ensure uniformity across all project files by standardizing headers and other metadata.
Efficiency: Save time by avoiding repetitive editing of template-generated content.
Professionalism: Customize file headers to include project-specific or company-specific details, boosting maintainability.
Where to find IDETemplateMacros.plist?
The IDETemplateMacros.plist file is not included in your project by default. You can create it manually in Xcode's UserData directory. The typical location is:
~/Library/Developer/Xcode/UserData/IDETemplateMacros.plist
Note: For a full list of possible locations visit Customize text macros Xcode's Help page.

If the file doesn't exist, you can create it using the following steps:
  1. Open Finder.
  2. Navigate to the ~/Library/Developer/Xcode/UserData directory.
  3. Create a new file named IDETemplateMacros.plist.
  4. Open the file and paste the following, then save it (no need to restart Xcode):
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>FILEHEADER</key>
    <string>
    //  Copyright © ___YEAR___ ___ORGANIZATIONNAME___. All rights reserved.
    //</string>
    <key>ORGANIZATIONNAME</key>
    <string>Vitalii Ivashchenko</string>
</dict>
</plist>
 ___ORGANIZATIONNAME___: The name for your organization that appears in boilerplate text throughout your project folder.
 ___YEAR___: The localized year string for the current year.

Note: For a full list of possible macros please visit Text macros reference Xcode's Help page.
